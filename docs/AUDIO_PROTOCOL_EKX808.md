# 得胜 TAKSTAR EKX-808 通信协议参考

> 8×8 数字矩阵音频处理器 · 32 位 DSP · 96kHz / 24bit
> 数据源: 得胜官方说明书 `EKX-808说明书_220421.pdf` (2022-04-28 版)
> 用途: PWA 中控平板 → NestJS 后端 → EKX-808 双向控制

---

## 1. 物理接口

| 接口 | 参数 | 用途 |
|---|---|---|
| **以太网 RJ45** | TCP/IP, 默认 IP 可在 PC 软件设定 | ✅ **我们项目使用** (走交换机, 不需要 USR-TCP232) |
| RS232 | 9600 / 8 / N / 1, DB9 (2=TXD, 3=RXD, 5=GND) | 备用 |
| RS485 | 9600 / 8 / N / 1, 级联 254 台 | 多 DSP 场景 |
| USB Type-B | PC 软件调试连接 | 仅调试 |

> 本项目部署: **DSP IP = `192.168.50.61`**, **DeviceAddr = 1** (单机, 不级联)。
> 直接 TCP 长连接或短连接均可, 建议短连接 (一问一答, 不维护会话状态)。

## 2. 帧格式

固定 9 字节, **无校验和** (用首尾 `7B 7D ... 7D 7B` 边界识别):

```
┌──────┬──────┬─────────────┬─────┬───────┬───────┬───────┬──────┬──────┐
│  0   │  1   │      2      │  3  │   4   │   5   │   6   │  7   │  8   │
├──────┼──────┼─────────────┼─────┼───────┼───────┼───────┼──────┼──────┤
│ DLE  │ STX  │ DeviceAddr  │ CMD │ Data1 │ Data2 │ Data3 │ STX  │ DLE  │
│ 0x7B │ 0x7D │  1 ~ 254    │ 40+ │  ??   │  ??   │  ??   │ 0x7D │ 0x7B │
└──────┴──────┴─────────────┴─────┴───────┴───────┴───────┴──────┴──────┘
```

**包头**: `0x7B 0x7D` (DLE + STX)
**包尾**: `0x7D 0x7B` (STX + DLE)
**DeviceAddr**: `0x01 - 0xFE` (固定本项目 = 0x01)

## 3. 命令字一览 (32 条, 零错印)

### 3.1 预设管理
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x40` | 调用预设矩阵 | `0=Factory / 1=User` | `0~12` | `0x00` | 矩阵预设 |
| `0x43` | 调用预设 | `0=Factory / 1=User` | `0~12` | `0x00` | 整机预设 |
| `0x4A` | 读取当前预设号 | `0x30` | `0x00` | `0x00` | 返回 1 字节 (0x00-0x0C = F00/U01-U12) |

### 3.2 增益/音量控制
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x41` | 增益 +/- | `0=In / 1=Out` | `0x00-0x0F` (16 路) | `+:0, -:1` | 步进 1 |
| `0x44` | **输入音量绝对值** | `0x00-0x0F` 通道 | HI-VOL | LO-VOL | -60~+12dB (见 §4) |
| `0x45` | **输出音量绝对值** | `0x00-0x0F` 通道 | HI-VOL | LO-VOL | 同上 |
| `0x54` | 音量增减 (统一) | `0=In / 1=Out` | `0x00-0x0F` | `+:0, -:1` | -60~-20: 2dB 步, -20~+12: 1dB 步 |
| `0x48` | **读取增益** | `0=In / 1=Out` | 通道 | `0x00` | 返回 3 字节 |

**读取增益返回值** (3 Byte):
- 1st: In/Out
- 2nd: Channel
- 3rd: 0-80 = -60~-20dB (0.5dB/Step), 80-280 = -20~0dB (0.1dB/Step), 280-400 = 0~+12dB (0.1dB/Step)

### 3.3 静音控制
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x42` | 静音 | `0=In / 1=Out` | 通道 | `0=No, 1=Yes` | |
| `0x49` | **读取静音状态** | `0=In / 1=Out` | 通道 | `0x00` | 返回 0/1 |

### 3.4 矩阵路由 (核心功能!)
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x4E` | **矩阵控制** | Channel (Out 0-7) | In (0-7) | `On:1 / Off:0` | Out N ← In M 通路 |
| `0x4F` | 读取单点矩阵 | Channel (Out) | In | `0x00` | 返回 0/1 |
| `0x61` | **读取全矩阵** | `0x00` | `0x00` | `0x00` | 返回 **160 字节** (5×32) |

### 3.5 编组控制 (整机层级)
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x46` | 编组音量 | `0=In / 1=Out` | `0-100` (百分比) | `0x00` | |
| `0x47` | 编组增益 +/- | `0=In / 1=Out` | `+:0, -:1` | `0x00` | |
| `0x4C` | 编组静音 | `0=In / 1=Out` | `0=No, 1=Yes` | `0x00` | |
| `0x4B` | 读取编组参数 | `0=In / 1=Out` | `0x00` | `0x00` | 返回 2 字节 (音量+静音) |

### 3.6 群组控制 (4 个独立群组, 可绑墙面板)
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x66` | 群组音量 | Group `0-7` | `0-100` | `0x00` | |
| `0x67` | 群组增益 +/- | Group `0-7` | `+:0, -:1` | `0x00` | |
| `0x63` | 读取群组参数 | Group | `0x00` | `0x00` | 返回 2 字节 |

### 3.7 AUX 通道 (效果器 + 自动混音)
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x51` | AUX 增益 +/- | `0x02` | `0x00` | `+:0, -:1` | |
| `0x52` | AUX 静音 | `0x02` | `0x00` | `0=No, 1=Yes` | |
| `0x53` | AUX 音量绝对值 | `0x02` | HI-VOL | LO-VOL | |
| `0x55` | **辅助开关** ★ | `0x02` | `0=效果, 1=摄像头, 2=自动混音, 3=AEC, 4=降噪` | `0=Off, 1=On` | |
| `0x56` | **AUX 输入选择** | `0=AUX, 1=Camera, 2=AutoMix` | Ch 16~9 位掩码 | Ch 8~1 位掩码 | 选哪些 In 混入 AUX |
| `0x57` | 输入反馈抑制 | `0x00` | 通道 | `0=OFF, 1-4=Level` | |
| `0x58` | 读取 AUX 增益 | `0x02` | `0x00` | `0x00` | |
| `0x59` | 读取 AUX 静音 | `0x02` | `0x00` | `0x00` | |
| `0x5B` | 读取辅助开关 | `0x02` | Select (0-4) | `0x00` | 返回 0/1 |
| `0x5C` | 读取 AUX 输入选择 | `0x02` | `0x00` | `0x00` | 返回 1 字节 Matrix |
| `0x5E` | 读取反馈抑制 | `0x00` | 通道 | `0x00` | 返回 Channel + Level |

### 3.8 电平表 (实时监控)
| CMD | 名称 | Data1 | Data2 | Data3 | 备注 |
|---|---|---|---|---|---|
| `0x4D` | **读取实时电平** | `0=In / 1=Out / 2=Aux` | 通道 | `0x00` | 返回 3 字节 (-128~+127dB = 0x80~0xFF/0x00~0x7F) |

## 4. 音量码值换算 (HI-VOL/LO-VOL)

输入/输出音量 (`0x44 / 0x45 / 0x53`) 用 2 字节 (HI-VOL + LO-VOL) 表示, 范围 0-400:

| 码值 | 实际 dB | 步长 |
|---|---|---|
| 0 - 80 | -60 ~ -20 dB | 0.5 dB/Step |
| 80 - 280 | -20 ~ 0 dB | 0.1 dB/Step |
| 280 - 400 | 0 ~ +12 dB | 0.1 dB/Step |

**0.0 dB 标定** = 码值 280 = `HI=0x01, LO=0x18` (0x118 = 280)
**官方范例** `In1 音量 +0.0dB`: `7B 7D 01 44 00 01 18 7D 7B`

## 5. 范例报文 (官方手册)

| 操作 | 报文 (Hex) |
|---|---|
| 调用用户预设 U02 矩阵 | `7B7D 01 40 01 02 00 7D7B` |
| 提升 In1 增益 | `7B7D 01 41 00 00 00 7D7B` |
| 不静音 Out1 | `7B7D 01 42 01 00 00 7D7B` |
| In1 音量设 0.0dB | `7B7D 01 44 00 01 18 7D7B` |
| Out2 音量设 -3.0dB | `7B7D 01 45 01 00 FA 7D7B` |
| 输入编组音量 90% | `7B7D 01 46 00 5A 00 7D7B` |
| 读取 In1 音量 | `7B7D 01 48 00 00 00 7D7B` |
| 读取当前预设 | `7B7D 01 4A 00 00 00 7D7B` |
| Out4 矩阵 In2 开 | `7B7D 01 4E 03 01 01 7D7B` |
| 读取 In5 反馈抑制 | `7B7D 01 5E 00 04 00 7D7B` |

## 6. PWA 中控集成路径

```
PWA 平板 (Vue3 + PWA)
  ↓ HTTP / WebSocket
NestJS 后端
  ↓ AudioModule
  ↓ EkxDspAdapter (待实现)
  ↓ TcpClient (复用 backend/src/adapters/transports/tcp-client.ts)
  ↓ TCP 短连接
EKX-808 DSP @ 192.168.50.61
```

### 集成点位
| 中控 UI | EKX-808 命令 | 备注 |
|---|---|---|
| 一键场景 (接待/路演/会议/夜间...) | `0x40` 调用矩阵预设 + `0x43` 调用预设 | DSP 预设里**预先**编辑好场景, PWA 只切预设号 (1 命令搞定 8 路) |
| 各分区音量滑块 (8 路) | `0x45` 输出音量绝对值 | 滑 0-100% → 映射到码值 0-400 |
| 各分区静音 | `0x42` 静音 + `0x49` 读取 | 双向 |
| 全场静音 / 全场降低 | `0x46` 编组音量 | 整层级一刀切 |
| 实时电平表 UI | `0x4D` 读电平 (1Hz 轮询) | 8 In + 8 Out 电平条 |
| 会议自动跟拍触发 | `0x55` 辅助开关 (Camera On) | 自动混音 + AEC + 摄像头跟踪 |
| 讲解员激活监控 | `0x5C` 读取 AUX 输入选择 | 看 AutoMix 当前选了哪路 |
| 反馈抑制等级 | `0x57` 输入反馈抑制 (LEVEL 1-4) | 各 In 通道独立 |

### 推荐工作流
1. **PC 端调音**: 现场调试时, 厂家工程师用 PC 软件 `TAKSTAR EKX-808 DSP Processor Editor V1.1` (USB 连机) 把所有 EQ / 压限 / 矩阵路由 / 编组定义好, 保存为用户预设 U01-U12
2. **场景预设**: U01 = 早班接待, U02 = 路演, U03 = 大型发布会, U04 = 会议, U05 = 夜间静音...
3. **PWA 只需切预设**: 一键按场景按钮 → 后端发 1 条 `0x40 + 0x43` → DSP 一秒切换所有 8 路设置
4. **微调走 PWA**: 临时音量微调用 PWA 滑块, 不动预设本身

## 7. 与方案 1 (HUSHAN XY260D) 协议对比

| 维度 | XY260D (方案 1, **已淘汰**) | **EKX-808 (方案 2)** |
|---|---|---|
| 帧格式 | 5 字节 + 校验和 | 9 字节, 边界包 |
| 校验印错 | **17 处** ❌ | 协议无错 ✅ |
| 命令总数 | 26 | **32** ✅ |
| 双向读取 | 8 条 | **11 条** ✅ |
| 接口 | 仅 RS232 | RS232 / RS485 / **TCP-IP** ✅ |
| 矩阵控制 | ❌ | **8×8 任意路由** ✅ |
| 预设管理 | ❌ | **25 个 (F00-F12 + U01-U12)** ✅ |
| 群组控制 | ❌ | **4 群组 + 编组** ✅ |
| AEC/摄像跟踪 | ❌ | ✅ |
| 设备地址 | 单台 | **1-254 级联** |
