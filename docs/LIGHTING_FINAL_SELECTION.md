## 1. LED 调光技术速查 (恒压 vs 恒流, 5 分钟看懂)

| 维度 | 恒压驱动 (CV) | 恒流驱动 (CC) |
| --- | --- | --- |
| 输出 | 固定电压 12V/24V/48V | 固定电流 350/700/1050mA |
| 适用灯具 | **LED 灯带**, LED 模组 (并联) | **LED 筒灯/射灯** (串联), 高功率 LED |
| 单台功率 | 主流 100-500W, 最大 1200W | 主流 30-200W |
| 调光方式 | **PWM** (高频通断, 信号 0-10V 或 DALI) | **0-10V** 或 DALI 改变电流 |
| 优势 | 长距离布线灵活, 多组并联 | 大功率单灯效率高 |
| 缺点 | 长距离压降, 远端变暗 | 不能并联 |

**LED 调光信号**:
- **DALI** (Digital Addressable Lighting Interface): 数字总线, 每灯/每驱动一个地址 (1-64), 支持分组、单点寻址、调光曲线、状态回读
- **0-10V**: 模拟信号, 简单, 0V=关 / 10V=最亮
- **TRIAC**: 可控硅切相, 老式, 对 LED 兼容差
- **PWM**: 高频通断, 内部信号, 通常由前三种之一驱动产生

**本项目策略**:
- LED 灯带 → DALI 数字输入 → 恒压电源内置 PWM 输出 → 灯带
- LED 灯具 (筒/射/吸顶) → DALI 数字输入 → 转 0-10V 模拟信号 → 灯具内部驱动控制亮度

## 6. 走廊 130m 灯带特别说明 (用户已确认方案: 不拆)

用户选择"不拆 + 大功率串接". 具体做法:

```
    AC 220V                          DC 24V
      │                                │
    电闸 → 驱动1 (HLG-480H-24DA) ────────┐
            │                            │ 并到同一段
    电闸 → 驱动2 (HLG-480H-24DA) ────────┤  130m 灯带
            │                            │ (同 DALI 地址同步调光)
    电闸 → 驱动3 (HLG-480H-24DA) ────────┘
            │
            └─ DALI 总线 (3 个驱动设同一地址 e.g. addr 4-6 同组 4)
```

**接线注意**:
1. 3 个驱动 220V 输入端各走自己的断路器 (4A 一路)
2. 3 个驱动 DC 24V 输出端**并联**到 130m 灯带 (灯带 + 接 +, - 接 -)
3. 3 个驱动的 DALI 输入并联到同一根 DALI 总线
4. 把 3 个驱动设成**同一 DALI 地址** (或同一 Group), 调亮度命令同步生效
5. 灯带长度太长压降明显时, 在 65m 处再注入一组 24V (即驱动 2 接到中点, 驱动 1 接头部, 驱动 3 接尾部)

**风险提示**:
- 24V × 54A 长距离压降大: 尾部可能比头部暗 20%+
- 建议**中段注入**方案 (3 个注入点)
- 如果调试时尾部明显暗, 还是要拆段

## 7. F101-R7 混合回路特别说明 (灯带 + 格栅)

R7 同一回路里同时有 LED 灯带 80m 和格栅长条灯 35m, 是两种不同的灯:
- **灯带**: 24V 恒压, 走 DALI 电源 (addr 7-8)
- **格栅**: 假设 AC 220V 一体化 LED, 走 0-10V 调光款 (addr 18)

**控制层**:
- 后端把 addr 7, 8, 18 配到同一 DALI Group (例: G3 走廊)
- 平板"走廊调亮度"命令 → DALI Group 3 → 3 个驱动同步响应
- 灯带和格栅就会同步亮/暗

**接线**:
- AC 220V 端: 仍是同一回路 (一个空开), 但驱动分两套
- DALI 总线: 通过, 灯带电源 + 格栅 0-10V 都接 DALI 网关同一总线

## 9. 给灯具采购的关键 spec

### 9.1 LED 灯带 (392m, 共 8 路)
- 输入电压: **DC 24V**
- 功率: **10W/m**
- 单段最长 5m (含 IC 防压降)
- 防尘 IP20 (吊顶内) / IP54 (走廊明装)
- 颜色温度: 暖白 3000K (前厅/展示) / 中性 4000K (走廊/办公)

### 9.2 LED 筒灯/射灯 (10 路以上, ~150 盏)
- 输入: AC 220V
- 调光接口: **0-10V 调光** (必须明确写入采购合同!)
- 功率: 筒灯 15-30W, 射灯 10-20W
- 颜色温度: 暖白 3000K (前厅) / 自然光 4000K (其他)
- 显色指数 (CRI) ≥ 80

### 9.3 LED 吸顶灯 (3 路 / 44 盏)
- 输入: AC 220V
- 调光接口: **0-10V**
- 功率: 50-60W
- 灯具尺寸/外观: 按装修风格选

### 9.4 灯箱 (12㎡)
- LED 模组: DC 24V 恒压
- 功率密度: 50W/㎡ (约 600W 总)
- 走 DALI 大功率电源 (HLG-600H)
