故障排查 — 平板/控制端

现象 1 平板打不开 http://192.168.50.10/control/
  1. 看主控机 (1F 弱电机柜内) 电源指示灯是否常亮
  2. 看主控机后部 两根网线都插好, 网卡指示灯闪烁
  3. 平板/手机自身网络是否正常 (WiFi 显示正常)
  4. 仍不行 → 联系下方驻场工程师
现象 2 某区灯光 / 空调 / 音响打不开
  1. 看对应设备本体是否物理通电 (有指示灯)
  2. 平板上重试 1-2 次
  3. 后台 (/admin/) → 设备管理 → 查对应设备状态 (离线 / 在线)
  4. 仍异常 → 联系下方驻场工程师

故障排查 — 主控机

现象 3 主控机死机 (远程也连不上, 平板全黑)
  1. 到 1F 弱电机柜找到 GK9000 主机
  2. 长按电源键 10 秒强制关机, 等指示灯熄灭
  3. 等 30 秒, 再短按电源键开机
  4. 等待 2 分钟, 系统自动启动 (会自动登录 + 自动恢复服务)
  5. 平板试访问 http://192.168.50.10/control/
  6. 仍不行 → 立即联系驻场工程师

火警 / 紧急情况

现象 4 火警 / 安全事故 / 人员紧急情况
  1. 优先按消防 / 安全流程处置, 不要管中控系统
  2. 中控会随建筑断电自动安全关停, 无需额外操作
  3. 事后恢复供电, 系统会自动重启 (按 §现象 3 步骤恢复)
