## 0. 一句话说清

**原设计有 4 个回路把"射灯 + 筒灯"绑在同一根 0-10V 信号线上, 无法独立调光, 现整改为"射灯 / 筒灯 分别独立调光"**, 需要:
- ➕ 加购 **1 台 雷特 LT-84A** (DALI→0-10V 转换器, ¥600)
- 🔧 把 4 个混合回路的灯具 0-10V 信号线**拆成两组分别接线**
- 📋 整改后 LT-84A 总数 **3 台 → 4 台**, 0-10V 通道 **12 路 → 16 路**

---

## 1. 问题描述

### 1.1 原设计 (错误)

LT-84A 共 3 台 / 12 路 0-10V 输出, 其中 **4 路混了射灯和筒灯**:

| LT-84A 通道 | 回路 | 筒灯 | **射灯** | 现状 |
|---|---|---|---|---|
| LT-84A-1 CH1 (DALI Addr 14) | F101-R2 | 9 只 | **13 只** | ⚠ 混在同一信号线 |
| LT-84A-1 CH3 (DALI Addr 16) | F101-R5 走廊 | 29 只 | **10 只** | ⚠ 混在同一信号线 |
| LT-84A-2 CH2 (DALI Addr 19) | F103-R1 | 7 只 | **12 只** | ⚠ 混在同一信号线 |
| LT-84A-3 CH1 (DALI Addr 22) | F201-R3 | 16 只 | **3 只** | ⚠ 混在同一信号线 |
| | | **61 只** | **38 只** | 4 个混合回路 |

### 1.2 这样做导致的功能缺失

| 业务场景 | 现在做不到 |
|---|---|
| **演示展品** — 周围筒灯调到 30%, 射灯保持 100% 聚光打展品 | ❌ 调一起调, 要暗一起暗 |
| **重点照明模式** — 关大部分筒灯, 只留射灯打装饰墙 | ❌ 射灯被筒灯绑死 |
| **节能巡场** — 留 1/3 筒灯, 关全部射灯 | ❌ 一个滑块控不开 |
| **会议接待** — 主席台射灯加强, 听众区筒灯调中 | ❌ 同回路绑死 |

业主已确认这些场景**都是必须的功能**, 因此设计必须整改。

## 3. 施工方具体改动 (★ 重点!)

### 3.1 物料增加

```
+ 雷特 TECH LT-84A × 1 台
+ RVV 2×0.5mm² 0-10V 信号线 × ~50m (4 个射灯组的独立信号线)
+ DIN 35mm 卡轨空间 × 1 个位 (主控柜 / 1F 弱电箱)
+ 0-10V 端子排 × 4 组
```

### 3.2 安装位置
**LT-84A-4** 装在 **1F 主控室弱电柜** (跟 LT-84A-1/2/3 同柜, 同一根 DALI 总线挂上去)。

### 3.3 接线变更 — 4 个回路 (★ 关键)

每个混合回路, **AC 220V 强电不变**, 仅 **0-10V 信号线分两组**:

#### F101-R2 (重点照明展示区)
```
原: LT-84A-1 CH1 (Addr 14) → 0-10V 信号 → 22 只灯具 (9 筒 + 13 射)

改:
  LT-84A-1 CH1 (Addr 14) → 0-10V 信号 A → 9 只筒灯
  LT-84A-4 CH1 (Addr 26) → 0-10V 信号 B → 13 只射灯  ★ 新拉一对信号线
```

#### F101-R5 (1F 走廊)
```
原: LT-84A-1 CH3 (Addr 16) → 0-10V 信号 → 39 只灯具 (29 筒 + 10 射)

改:
  LT-84A-1 CH3 (Addr 16) → 0-10V 信号 A → 29 只筒灯
  LT-84A-4 CH2 (Addr 27) → 0-10V 信号 B → 10 只射灯  ★ 新拉一对信号线
```

#### F103-R1 (综合展销)
```
原: LT-84A-2 CH2 (Addr 19) → 0-10V 信号 → 19 只灯具 (7 筒 + 12 射)

改:
  LT-84A-2 CH2 (Addr 19) → 0-10V 信号 A → 7 只筒灯
  LT-84A-4 CH3 (Addr 28) → 0-10V 信号 B → 12 只射灯  ★ 新拉一对信号线
```

#### F201-R3 (2F 服务中心)
```
原: LT-84A-3 CH1 (Addr 22) → 0-10V 信号 → 19 只灯具 (16 筒 + 3 射)

改:
  LT-84A-3 CH1 (Addr 22) → 0-10V 信号 A → 16 只筒灯
  LT-84A-4 CH4 (Addr 29) → 0-10V 信号 B → 3 只射灯  ★ 新拉一对信号线
```

### 3.4 接线示意 (单回路通用)

```
                                  AC 220V (空开后)
                                       │
                                       ├──┬──┬──┬──┬──┬──┬─...
                                       ↓  ↓  ↓  ↓  ↓  ↓
                                      筒1 筒2 筒3 ... 射1 射2 ...

  LT-84A-1 CH1 (Addr 14):
   ┌─0-10V+─┐
   │        │              ┌─→ 筒1 0-10V+
   │        ├──── 信号 A ───┼─→ 筒2 0-10V+
   └─0-10V-─┘              │  ...
                            └─→ 筒9 0-10V+
                            (9 只筒灯共用)

  LT-84A-4 CH1 (Addr 26):   ★ 新加的
   ┌─0-10V+─┐
   │        │              ┌─→ 射1 0-10V+
   │        ├──── 信号 B ───┼─→ 射2 0-10V+
   └─0-10V-─┘              │  ...
                            └─→ 射13 0-10V+
                            (13 只射灯共用)
```

**关键点**: 同一组灯具 (例如全部 13 只射灯) 的 0-10V 信号线**并联**到 LT-84A 同一通道, 筒灯和射灯各走各的一对线, **物理隔离**。

### 3.5 信号线规格
- 类型: RVVP 2×0.5mm² 屏蔽双绞 (建议) 或 RVV 2×0.5mm² (低端)
- 距离: 单通道 0-10V 信号线总长 ≤ 100m
- 颜色: 建议红+黑 (信号线统一颜色, 跟强电区分)
- 标识: 每段进灯具前贴标签 "0-10V CH1 / 筒灯组" 或 "0-10V CH4 / 射灯组"

### 3.6 灯具采购需求 (★ 同时再次强调)
所有筒灯、射灯、吸顶灯、格栅灯**必须是 0-10V 调光款** (LED 一体化驱动, 带 PURPLE/GREY 0-10V 输入线)。如订货时漏选 0-10V 款, 整个系统无法调光, **此时回不去**。
