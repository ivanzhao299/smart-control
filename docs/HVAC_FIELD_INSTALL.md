# 中央空调系统现场施工 SOP

> 金湖展贸中心智能控制系统 · 中央空调子系统现场安装与调试指引
> 适用于: 奥克斯商用 VRF 外机 **DLR-735W5/DCM-ARVX7(II)** + **CCM-270B** Modbus TCP 网关
> 内机数量: **10 台** (本项目实际部署), CCM-270B 上限 64 台
> 估时: 通讯调试 2-3 小时 (外机/内机安装由空调厂家完成, 不在此列)

## 2. 系统拓扑

```
┌────────────────┐    以太网    ┌──────────────────┐
│ 中控主机       │ ──────────── │ 弱电机柜交换机   │
│ 192.168.50.10  │              │                  │
└────────────────┘              └──────────────────┘
                                          │
                                          │ 以太网 (RJ45)
                                          ▼
                                ┌──────────────────────┐
                                │ CCM-270B 网关         │
                                │ 192.168.50.50:502    │
                                │ DC 12V/1A 适配器供电  │
                                └──────────────────────┘
                                          │
                              XYE 总线 (3 线: X/Y/E)
                                          │
        ┌─────────────────────┬──────────┴───────────┬──────────────────┐
        ▼                     ▼                      ▼                  ▼
   ┌────────┐            ┌────────┐             ┌────────┐          ┌────────┐
   │ IDU #1 │            │ IDU #2 │   ……        │ IDU #9 │          │ IDU #10│
   │ 一层前厅│           │ 一层路演│             │ 二层接待│         │ 二层指挥│
   └────────┘            └────────┘             └────────┘          └────────┘
                                          │
                                          │ XYE 接到外机
                                          ▼
                                ┌──────────────────────┐
                                │ DLR-735W5 外机       │
                                │ 楼顶设备平台         │
                                └──────────────────────┘
```

**关键点**:

- CCM-270B 走 **以太网 (Modbus TCP)** 上行到中控, 不需要 RS485 转换器 (跟 DALI 不一样)
- CCM-270B 走 **XYE 总线** 下行到外机 / 内机 (这是奥克斯私有冷媒侧通讯)
- XYE 总线 = 3 线: X (+) / Y (-) / E (屏蔽地), 必须用 RVVP 屏蔽双绞线, 屏蔽层只在一端接地

## 3. 内机地址规划

> CCM-270B 用**寄存器偏移**寻址内机, 每台内机占用一个 16-寄存器 块.
> 内机的物理地址 (1-64) 由空调厂家在内机控制板上拨码或在内机线控器中设置.

| indoorIdx | 寄存器基址 | 楼层 | 区域 | 内机编号 (现场拨码) |
| --- | --- | --- | --- | --- |
| 1  | 0x0000 | 1F | 前厅 / 园区展示       | IDU-01 |
| 2  | 0x0010 | 1F | 路演 / 洽谈区          | IDU-02 |
| 3  | 0x0020 | 1F | 企业展位区 (F102)     | IDU-03 |
| 4  | 0x0030 | 1F | 综合展销区 (F103)     | IDU-04 |
| 5  | 0x0040 | 1F | 物贸交易展示区 (F104) | IDU-05 |
| 6  | 0x0050 | 2F | 前厅 / 走廊            | IDU-06 |
| 7  | 0x0060 | 2F | 企业服务中心           | IDU-07 |
| 8  | 0x0070 | 2F | 共享办公               | IDU-08 |
| 9  | 0x0080 | 2F | 产业研究 / 接待        | IDU-09 |
| 10 | 0x0090 | 2F | 运营指挥中心           | IDU-10 |

**重要**: 空调厂家拨码完成后, 现场必须**逐台核对** indoorIdx ↔ 实际位置, 否则后台开"1F前厅"会启动错误的内机.

## 4. CCM-270B 网关接线

### 4.1 端子定义 (背板)

| 端子 | 含义 | 接线 |
| --- | --- | --- |
| L / N | AC 220V 或 DC 12V 输入 | (按机型, 看铭牌) |
| X / Y / E | XYE 通讯总线 (接外机+内机) | RVVP 3×0.75 |
| RJ45 | 以太网口 (上行 Modbus TCP) | Cat5e → 交换机 |
| RST | 复位按钮 | (出厂默认, 不要按) |
| DIP | 拨码开关 | 见 4.3 |

### 4.2 接线步骤

1. **断电**: 确认 220V 总闸关闭
2. 接 XYE 线:
   - 网关 X → 外机 X → 所有内机 X (并联, 不要星型)
   - 网关 Y → 外机 Y → 所有内机 Y
   - 网关 E → 外机 E (屏蔽地, **不要接到内机**, 只外机一端接地)
3. 接电源: 按铭牌接 L/N 或 DC 12V
4. 接网线: RJ45 → 弱电交换机
5. **送电前**复核线序, 万用表量 X-Y 之间电阻 (开路应 > 100Ω)

### 4.3 拨码 (DIP)

| 位 | 功能 | 出厂默认 | 本项目设置 |
| --- | --- | --- | --- |
| 1 | Modbus 地址 (slaveId) bit 0 | ON | ON |
| 2 | Modbus 地址 bit 1 | OFF | OFF |
| 3 | Modbus 地址 bit 2 | OFF | OFF |
| 4 | Modbus 地址 bit 3 | OFF | OFF |
| 5 | 波特率 (TCP 用不到, 走以太) | — | — |
| 6 | XYE 终端电阻 (120Ω) | ON (链尾) | **ON** (本机是 XYE 链头) |

→ slaveId = 0001 = **1** (与 `.env` `HVAC_DEFAULT_SLAVE_ID=1` 一致)

> 拨码组合 = 二进制值 + 1 (有的型号是 +0, 现场按说明书). slaveId 范围 1-15.

## 5. IP 配置

CCM-270B 默认 IP 出厂常见值: `192.168.1.200` 或 `192.168.0.100`.

**改成项目 IP 192.168.50.50** 的方法 (优先级从高到低):

**方法 A** (推荐, 厂家提供): 用奥克斯随机的 **CCM-270B 配置工具** (Windows)
1. 笔记本网卡临时改到 192.168.1.x 段
2. 网线直连网关 RJ45
3. 工具自动发现网关, 改 IP/掩码/网关
4. 重启网关, 笔记本网卡恢复正常

**方法 B** (无配置工具, 网页登录): 部分批次支持 HTTP 配置页
- 浏览器访问 `http://192.168.1.200`
- 默认账号 `admin` / `admin`
- 网络设置 → 改静态 IP

**方法 C** (DHCP 应急): 把网关接入项目交换机后, 在路由器 DHCP 列表里找它 MAC (奥克斯 OUI `8C:1F:64` / `D8:80:39` 等), 把它绑定到 192.168.50.50.

## 6. Modbus Poll 寄存器调试

> **必做!** 在跟中控联调之前, 先用 Modbus Poll 工具直接读写寄存器, 确认协议是否对.

### 6.1 Modbus Poll 连接

1. File → New
2. Connection → Connect:
   - Connection: **Modbus TCP/IP**
   - IP: `192.168.50.50`
   - Port: `502`
   - Slave ID: `1`
3. Slave Definition:
   - Function: **03 Read Holding Registers**
   - Address: `0` (1 号内机)
   - Quantity: `16`
   - Scan Rate: `1000 ms`

### 6.2 测试用例

| 测试 | 操作 | 预期寄存器变化 |
| --- | --- | --- |
| 读 1 号内机状态 | FC03, 地址 0, 数量 16 | 16 个寄存器返回, 0/1 寄存器是开关与模式 |
| 开 1 号内机 | FC06, 地址 0, 值 1 | 1 号内机风机启动 (3-5 秒延迟) |
| 设定温度 25℃ | FC06, 地址 2, 值 250 | 内机线控器显示 25℃ |
| 切换制冷模式 | FC06, 地址 1, 值 0 | 内机线控器显示"制冷" |
| 读 5 号内机状态 | FC03, 地址 64 (0x40), 数量 16 | 5 号内机状态 |
| 关 1 号内机 | FC06, 地址 0, 值 0 | 1 号内机风机停 |

> **重要**: 现场拿到 CCM-270B 实际说明书后, 如果 mode/fan 寄存器值与本表不一致 (例如奥克斯部分批次 mode 顺序是 cool=1/heat=2/fan=3/dry=4/auto=0), 修改:
> `backend/src/adapters/hvac/aux-ccm270b-registers.ts` 中的 `AUX_MODE_MAP` / `AUX_FAN_MAP` 即可.

## 7. 中控后台联调

### 7.1 后台配置

`backend/.env`:
```bash
MOCK_MODE=false                  # 切换到真实设备模式
HVAC_HOST=192.168.50.50
HVAC_PORT=502
HVAC_DEFAULT_SLAVE_ID=1
DEVICE_TIMEOUT_MS=3000
DEVICE_RETRIES=3
```

重启后端:
```bash
pm2 restart smart-control-backend
pm2 logs smart-control-backend --lines 100
```

期望看到: `HvacAdapter ready (mode=modbus-tcp)`

### 7.2 健康检查

```bash
curl -s http://localhost:3000/api/health/devices | jq '.data[] | select(.category=="hvac")'
```

期望: `status: "online"`

### 7.3 单台内机功能验证 (在 /control 控制台)

进入"空调"页面, 依次:

| 操作 | 预期 |
| --- | --- |
| 点击"一层前厅 / 园区展示" 开机 | 1F 前厅内机风机启动 |
| 设定温度 22°C | 卡片显示 22°C, 内机线控器显示 22°C |
| 切到制冷 | 内机模式指示灯切到制冷 |
| 切到高风 | 内机风机噪音明显变大 |
| 点击关机 | 风机停 |

依次完成 10 台内机, 标记结果到 UAT 表.

### 7.4 场景联调

执行"开馆模式" → 检查空调动作是否生效 (按场景 spec, 例如全部开机 + 设定 24°C + 制冷模式).

## 8. 常见故障排查

| 现象 | 可能原因 | 处置 |
| --- | --- | --- |
| 后台 `/api/health/devices` HVAC offline | 网关 IP 不对 / 网线松 | ping 192.168.50.50, 检查交换机端口 |
| ping 通但 Modbus 502 不通 | 网关防火墙 / 端口被改 | 查网关配置工具, 复位 |
| 全部内机控制不响应 | XYE 极性反接 / 终端电阻没开 | 量 X/Y 极性, 网关链头加 120Ω |
| 1 台内机不响应, 其他正常 | 该内机地址拨错 / XYE 线松 | 找空调厂家用 ACBT 软件复读地址 |
| 开机后内机风机响但不出冷 | 外机未运行 / 冷媒不足 | 找空调厂家排查 (智能化不负责冷媒) |
| 温度显示 `--` 或 65535 | 内机离线 / 温感故障 | 重启内机, 找空调厂家 |
| Modbus 异常码 0x02 | 寄存器地址越界 (内机不存在) | 核对 indoorIdx ≤ 实际内机数量 |
| Modbus 异常码 0x03 | 写入值越界 (温度超 16-30) | 后端已限制, 不应出现; 出现则查日志 |

## 10. 维护要点

- **XYE 总线接地**: 屏蔽层只在外机一端接地, 严禁两端接地形成接地环路 (会引入干扰导致丢包)
- **网关供电**: 跟内机/外机分开走 220V 回路, 防止压缩机启停瞬间影响网关
- **冷媒侧故障**: 智能化不负责诊断, 报警代码记录后转空调厂家
- **冬季制热**: 商用 VRF 制热效果会随外温下降, -5°C 以下需开启电辅助 (奥克斯线控器面板设置)
- **节能策略 (可选, 后期开发)**:
  - 闭馆模式: 全部关机
  - 夜间巡检模式: 维持 18°C 防潮
  - 高温日: 早 9 点自动预冷, 10 点开馆时已舒适
