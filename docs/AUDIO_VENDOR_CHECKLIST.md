## 4. 预设场景录入 (M3: 第 2-3 天, 核心交付!)

乙方必须在 EKX-808 的 **U01-U12 用户预设位**录好下表 **12 个场景**, 每个场景包含:
- 矩阵路由 (8×8)
- 各路输出音量 (dB)
- 各路输入增益 (dB)
- EQ 曲线 (各路独立)
- 静音/取消状态
- AUX 辅助开关状态 (AEC / NR / 自动混音 / 摄像跟踪)

### 4.1 预设场景定义

| U号 | 场景名 | 用途 | 关键设置 |
|---|---|---|---|
| **U01** | 早班接待 | 8:00-10:00 全场低音量 BGM | 8 路低音量 (-15dB), 输入 = MG-10M 定时器, 关讲解 |
| **U02** | 日常运营 | 10:00-17:00 标准音量 | 8 路 -10dB, BGM + 讲解员通道 (RSSI 自动激活) |
| **U03** | 路演演讲 (ZONE 1) | 前厅大屏路演 | ZONE 1 主声道, 其他区静音 / 低音量, AUX = 自动混音开 |
| **U04** | 大型发布会 | LED 大屏 + 主会场全开 | 所有区开, ZONE 1 (LED 大屏) +5dB 突出 |
| **U05** | 会议室独立 (ZONE 8) | 会议室开会, 其他区 BGM 不动 | ZONE 8 独立, AEC 开, NR 开, 反馈抑制 Level 3 |
| **U06** | 跟随讲解模式 | 讲解员带客户走动 | WTG-800 RSSI 自动激活当前区 |
| **U07** | 政府接待 (VIP) | 高规格接待, 提前热场 | 全场 +0dB, BGM 庄重 |
| **U08** | 夜间静音 | 18:00-次日 8:00 | 全场静音, 仅保留紧急广播路径 |
| **U09** | 应急广播 | 消防/疏散 | 全场最大音量, 强插紧急通道 |
| **U10** | 大屏视频独立 | 仅 LED 大屏播放视频, 周围安静 | ZONE 1 全频音箱 + SUB12, 其他 -∞ |
| **U11** | 自定义 1 | 备用 (预留, 默认 = U02) | — |
| **U12** | 自定义 2 | 备用 (预留, 默认 = U02) | — |

### 4.2 各场景录完后, 必须出一份《**预设配置交接表**》

乙方导出 EKX-808 预设文件 (`.ekx` 或类似格式) **交给甲方**, 内容包括:
- 每个 U01-U12 的矩阵路由表 (8×8 圈叉图)
- 每路输入/输出的 dB 值 + EQ 频率/增益/Q 值
- AUX 各开关状态
- AEC 参数 (会议室)

**没有这份交接表 = 预设没交付**。甲方靠这份表写 NestJS 代码调用对应预设号。

## 5. 音质调试 (M3: 第 2-3 天)

### 5.1 各分区 SPL 一致性
- [ ] 用声压计 (乙方自带) 在每区中心位置测量
- [ ] U02 标准音量场景下, 各区 SPL 在 **65-72 dBA** 范围内, 区间差 ≤ 3dB
- [ ] 拍照记录每区测量值, 附入交接表

### 5.2 EQ 调整 (每区独立)
- [ ] 各区频响曲线**目标**: 80Hz-12kHz 范围内 ±6dB 内
- [ ] LED 大屏区 (JBL): 80Hz-16kHz 范围内 ±3dB (要求更高)
- [ ] 会议室 (ZONE 8): 200Hz-8kHz 优先, 突出人声清晰度
- [ ] 走廊区: 截掉 200Hz 以下 (避免低频驻波)

### 5.3 反馈抑制 (无线麦区域)
- [ ] WTG-800T 在各区缓慢转动 + 提音量, 找出啸叫频点
- [ ] 配置 EKX-808 反馈抑制 Level 2-3 (各区独立)
- [ ] **演示**: 讲解员手持麦, 在 ZONE 1-7 任一区, 音量推至日常使用上限, **不啸叫**

### 5.4 会议室 AEC (ZONE 8 独立)
- [ ] 用 PC 软件进入 AUX → AEC Setting 调整 VCS IN / AEC OUT 电平
- [ ] 模拟视频会议: 远端讲话 + 本地讲话, 远端**听不到自己回音**
- [ ] NR 降噪开启, 现场风扇/空调声不入麦

### 5.5 压限保护
- [ ] 所有功放输出加 limiter @ -3dBFS 防爆音
- [ ] **测试**: DSP 输入推爆, 输出**不超 -3dBFS** (功放不损坏)

## 6. WTG-800 分区导览标定 (M3: 第 3 天)

### 6.1 频道选择
- [ ] 8 个接收器 + 1 个发射器**统一选同一频道** (23 选 1, 记录频道号 = _____)
- [ ] 接收器 ID 设置: WTG-Z1 = ID 1, WTG-Z2 = ID 2, ..., WTG-Z8 = ID 8

### 6.2 功率档位标定 (0-9 档)
乙方工程师**讲解员模拟走查** 8 个区, 每区在中心位置, 调整该区接收器功率档位:
- 走到 ZONE X 中心: WTG-ZX 应**在 2 秒内**接管音频, 其他区域**保持原状态**
- 走到 ZONE X 边缘 (邻区交界): 切换稳定, 不**反复跳变**
- 走出所有区 (室外/楼梯): 全部区域**回到原状态** (BGM 或静音)

### 6.3 8 区走查测试 (验收必做)
甲方代表**亲自佩戴 WTG-800T**, 按下表顺序走查, 每区记录:

| 区 | 走入后激活时间 | 走出后回归时间 | 邻区有无串音 | Pass/Fail |
|---|---|---|---|---|
| Z1 路演区 | ≤ 2s | ≤ 3s | 无 | ☐ |
| Z2 1F 前厅 | ≤ 2s | ≤ 3s | 无 | ☐ |
| Z3 1F 企业展示 | ≤ 2s | ≤ 3s | 无 | ☐ |
| Z4 1F 物贸展示 | ≤ 2s | ≤ 3s | 无 | ☐ |
| Z5 2F 前厅 | ≤ 2s | ≤ 3s | 无 | ☐ |
| Z6 2F 共享办公 | ≤ 2s | ≤ 3s | 无 | ☐ |
| Z7 2F 商务办公 | ≤ 2s | ≤ 3s | 无 | ☐ |
| Z8 会议室 | ≤ 2s | ≤ 3s | 无 | ☐ |

任意 1 区 Fail = WTG 标定未完成, 重调。

### 6.4 功率档位**交接表**
出一份《8 区 WTG-800R 功率档位表》, 标注每区设的档位 (0-9), 后期甲方运维参考。

## 7. 综合验收测试 (M4: 第 3 天)

### 7.1 预设切换全验
甲方在乙方笔记本上的 PC 软件**逐个调用 U01-U12**, 现场听:
- [ ] U01 早班接待: 音量低, BGM 正常播
- [ ] U02 日常: 音量适中
- [ ] U03 路演: ZONE 1 突出, 其他区低
- [ ] U04 大型发布: 全开, LED 区突出
- [ ] U05 会议室: ZONE 8 独立, 其他不变
- [ ] U06 跟随讲解: WTG-800T 走 8 区, 准确切换 (已在 §6.3 验)
- [ ] U07 VIP: 全场 +0dB, 庄重
- [ ] U08 夜间静音: 全场静音
- [ ] U09 应急广播: 全场最大, 强插
- [ ] U10 大屏视频: ZONE 1 全频 + SUB12 工作, 其他静默
- [ ] U11/U12 备用: 切换不报错

### 7.2 TCP 远程调用验证 (集成接口)
甲方现场用**笔记本接交换机**, 用串口/网络调试助手发送以下指令到 192.168.50.61:
- [ ] 发 `7B 7D 01 43 01 02 00 7D 7B` (调用 U02) → DSP 现场切换到 U02
- [ ] 发 `7B 7D 01 43 01 03 00 7D 7B` (调用 U03) → DSP 现场切换到 U03
- [ ] 发 `7B 7D 01 4A 30 00 00 7D 7B` (读当前预设) → 返回正确字节

**这一项关系到我们 PWA 远程控制能不能用, 必须现场验。**

### 7.3 应急测试
- [ ] 拔 EKX-808 网线 → 麦克风 / 调音台直接接的本地源**仍能出声** (备份路径)
- [ ] 关 EPO-802P 时序器 → 全部功放断电 (1 键应急关闭)
- [ ] 现场触发应急广播 → U09 场景生效, 全场最大声
