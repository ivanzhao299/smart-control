### 3.3 IP67 防护说明

防尘等级 (第 1 位 6 = 完全防尘): 灰尘无法进入

适合:
- 弱电机柜 / 公共电箱内 (IP65 + 已足够, IP67 是冗余)
- 走廊吊顶内 (有冷凝水风险时需 IP67)

不适合:
- 户外暴雨直淋 (需 IP68/IP69K)
- 油雾环境 (需特殊型号)

### 3.4 DALI 接线

每台 HLG-DA 都有 2 根 DALI 输入线 (无极性):
- DA+ / DA1
- DA- / DA2

**接线规则**:
- 16 台驱动 + 网关 = 17 个 DALI 节点 (远低于 64 上限)
- DALI 总线**菊花链或星型都可**, 无极性 (但建议固定 + / -)
- 最长 300m (距网关), 我们项目 ≤100m, 无压力
- 总线电缆: **RVVP 2×0.5mm² 屏蔽双绞**, 屏蔽层网关端一点接地

## 5. 大功率灯带电压压降分析 (3 个并联组)

### 5.1 F101-R4 走廊 130m / 1300W (3 台并联)

电流: 1300W / 24V = **54.2A**

如果**单点注入** (全部电流走单一馈线):
- 130m 灯带, 假设 16mm² 铜线
- 电阻: ρ × L / A = 0.0175 × 130 / 16 = 0.142 Ω
- 压降: 54.2A × 0.142 = **7.7V** ← 严重! 24V 系统压降 32%
- 远端电压: 24 - 7.7 = **16.3V** ← 灯带 60% 亮度

**3 点注入** (头 / 中 / 尾各一台 HLG-480 + 灯带分 3 段):
- 每段 43m, 433W, 电流 18A
- 单段压降: 18A × (0.0175 × 43 / 6mm²) = **2.3V** (10% 压降)
- 远端电压: 24 - 2.3 = **21.7V** ← 灯带 90% 亮度 (可接受)

**推荐做法**:
- 灯带末端接到中段, 灯带中段接到末段, 形成 "首尾相接" 减少压降
- 馈线截面 ≥ **6mm² 铜线** (单段)

### 5.2 F101-R7 灯带 80m / 800W (2 台并联)

电流: 800W / 24V = 33.3A

**2 点注入** (头 + 中):
- 每段 40m, 400W, 16.7A
- 单段压降 (6mm² 铜): 16.7A × (0.0175 × 40 / 6) = **1.95V** (8%)
- 远端电压: 22.05V ← 92% 亮度 ✓

### 5.3 F201-R4 灯带 85m / 900W (2 台并联)

电流: 900W / 24V = 37.5A

**2 点注入**:
- 每段 42m, 450W, 18.75A
- 单段压降 (6mm² 铜): 18.75A × (0.0175 × 42 / 6) = **2.3V** (10%)
- 远端: 21.7V ← 90% 亮度 ✓

| 馈线 (各规格小段) | 各 5m | 应急接线 |

### 7.2 关键易损件

- HLG 系列 PSU 寿命**约 7-10 年** (24/7 运行, 良好散热条件)
- LT-84A 寿命**> 10 年** (无运动部件)
- LED 灯带寿命 **30,000-50,000 小时** = 3-5 年 (每天 12h 算)
- 灯具 (筒灯/吸顶) 寿命 **25,000-50,000 小时**

### 7.3 故障替换流程

| 故障 | 现象 | 排查 | 处置 |
| --- | --- | --- | --- |
| HLG 单台挂 | 该回路全暗 | 量 DC 24V 输出 = 0 | 关 AC, 拔端子, 换新 HLG, 接线, 上电 |
| 一台 HLG 黄灯 | 过载/过温保护触发 | 看 LED 指示灯 + 工作温度 | 加散热, 减负载, 重启 |
| LT-84A 通道挂 | 某 0-10V 回路不响应 | 量该通道输出电压 | 整台换新 (4 通道一起换) |
| DALI 总线全挂 | 所有 DALI 设备无响应 | ping CY-DALI64A 网关, 量总线电压 | 查网关供电 / 总线断线 |

## 8. 采购询价点

跟供应商询价时, **必须明确**:

1. **型号准确性**: HLG-480H-24**DA** (-DA 是 DALI 版本, 不要买成 -A 的 0-10V 版本或不带后缀的固定输出)

3. **批次**: 同一批次保证电源参数一致 (并联同一灯带时尤其关键)
4. **保修**: MEANWELL 官方保 **5 年** (要发票 + 国行)
